# [6-3-0] Structure (champs personnalisés) des entités

## Description

Permettre à l'intégrateur de gérer les champs personnalisés des entités organisationnelles : visualiser les champs système, ajouter/modifier/archiver des champs personnalisés, et configurer des options avancées (commentaires sur transitions, auto-génération, formatage, règles de validation inter-champs).
