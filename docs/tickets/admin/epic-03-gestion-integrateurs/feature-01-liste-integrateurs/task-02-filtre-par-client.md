# [3-1-2] Filtre des intégrateurs par client

## Référence Figma

- [home-intégrateurs](https://www.figma.com/design/Iv0JzhpeVpx3fsKmw3E7Yu/Olympe?node-id=19-1203)

## Description

Permettre à l'Admin Delta de filtrer la liste des intégrateurs pour n'afficher que ceux assignés à un client donné.

## Détails fonctionnels

- Select déroulant positionné dans la barre d'outils du tableau, à droite de la recherche
- Option par défaut : "Tous les clients"
- Liste des clients triée alphabétiquement
- Sélectionner un client filtre le tableau pour n'afficher que les intégrateurs assignés à ce client
- Le filtre s'applique indépendamment de la recherche textuelle (les deux peuvent être combinés)
