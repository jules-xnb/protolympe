# [5-4-0] Structure (champs) d'un objet métier

## Description

Permettre à l'intégrateur de gérer la structure d'un objet métier : visualiser les champs système, ajouter/modifier/archiver des champs personnalisés, et configurer les types de champs avancés (référentiels, références entre objets, champs calculés, champs de référence).
