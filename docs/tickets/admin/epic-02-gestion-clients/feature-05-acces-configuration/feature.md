# [2-5-0] Accès à la configuration d'un client

## Référence Figma

- [home-clients](https://www.figma.com/design/Iv0JzhpeVpx3fsKmw3E7Yu/Olympe?node-id=3-4421)

## Description

Permettre à l'Admin Delta et à l'intégrateur de basculer vers le mode Intégrateur dans le contexte d'un client spécifique, afin d'accéder à son espace de configuration (entités, référentiels, objets métiers, etc.).
