# [5-5-0] Import d'objets métiers et de champs

## Description

Permettre à l'intégrateur d'importer des définitions d'objets métiers avec leurs champs via CSV, d'importer des champs dans un objet existant, et d'importer des instances d'un objet métier. Chaque import suit un wizard multi-étapes : upload, mapping, prévisualisation, résultats.
