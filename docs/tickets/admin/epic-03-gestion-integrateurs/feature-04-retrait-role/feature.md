# [3-4-0] Retrait du rôle d'intégrateur

## Référence Figma

- [home-intégrateurs](https://www.figma.com/design/Iv0JzhpeVpx3fsKmw3E7Yu/Olympe?node-id=19-1203)
- [home-admin-intégrateurs-snackbar](https://www.figma.com/design/Iv0JzhpeVpx3fsKmw3E7Yu/Olympe?node-id=88-8018)

## Description

Permettre à l'Admin Delta de retirer le rôle d'intégrateur à un utilisateur. L'utilisateur perd alors son accès à la plateforme en tant qu'intégrateur. Cette action nécessite une confirmation.
