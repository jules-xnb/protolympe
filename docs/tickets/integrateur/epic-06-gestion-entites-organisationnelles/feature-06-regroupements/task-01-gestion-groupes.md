# [6-6-1] Gestion des groupes d'entités

## Description

Créer, modifier et supprimer des regroupements d'entités organisationnelles.

## Détails fonctionnels

- Accessible via l'onglet "Regroupements" de la page des entités
- Liste des groupes avec :
  - Nom du groupe
  - Badge avec le nombre de membres
  - Groupes dépliables pour voir les membres
- Bouton "Créer un groupe" :
  - Dialog avec champs : nom (requis) et description (optionnel)
- Suppression d'un groupe :
  - Bouton de suppression par groupe
  - Dialog de confirmation avant suppression
  - La suppression du groupe ne supprime pas les entités membres

## Critères de done

- [ ] Liste des groupes avec compteur de membres
- [ ] Création de groupe (nom + description)
- [ ] Suppression avec confirmation
- [ ] Groupes dépliables
- [ ] La suppression ne touche pas les entités
