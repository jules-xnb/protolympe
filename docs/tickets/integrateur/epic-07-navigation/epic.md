# [7-0-0] Intégrateur - Navigation

## Description

Permettre à l'intégrateur de configurer la structure de navigation de l'espace utilisateur final d'un client. La page affiche une arborescence interactive de deux types d'éléments imbriqués : les **groupes** (conteneurs organisationnels) et les **vues** (pages configurables). L'intégrateur peut créer, renommer, dupliquer, activer/désactiver et supprimer ces éléments, les réorganiser par drag & drop hiérarchique, et configurer les permissions d'accès par rôle. Chaque vue dispose d'un **Page Builder** visuel (grille 12 colonnes, palette de blocs Structure et Métier) permettant de composer la page avec des blocs configurables : tableaux de données, organisation (EO), questionnaires, réponses questionnaires, utilisateurs, sections et séparateurs. Le builder supporte l'auto-save, la publication, et la configuration détaillée de chaque bloc (colonnes, filtres, pré-filtres, recherche, formulaire de création).
