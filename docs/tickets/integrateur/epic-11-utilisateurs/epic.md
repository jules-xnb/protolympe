# [11-0-0] Intégrateur - Utilisateurs

## Description

Permettre à l'intégrateur de gérer les utilisateurs finaux d'un client. Le tableau principal affiche les utilisateurs avec leur statut (Actif, Inactif, À configurer, Prêt à activer, Sans profil), profils assignés, date d'ajout et champs personnalisés. L'intégrateur peut **inviter** de nouveaux utilisateurs (prénom, nom, email + champs personnalisés), **activer/désactiver** leur compte, **assigner/retirer des profils-types**, et **modifier les champs personnalisés** directement dans un drawer de détails. La page propose un **import CSV** en 3 étapes (upload, aperçu/validation, exécution) avec auto-détection des colonnes et assignation de profils. L'export CSV est disponible. Une sous-page dédiée permet de **gérer les définitions de champs personnalisés** utilisateur (texte, email, nombre, date, booléen, sélection, initiales) avec import/export CSV et archivage.
