# [2-1-2] Recherche dans la liste des clients

## Référence Figma

- [home-clients](https://www.figma.com/design/Iv0JzhpeVpx3fsKmw3E7Yu/Olympe?node-id=3-4421)

## Description

Permettre à l'Admin Delta et à l'intégrateur de filtrer les clients affichés dans le tableau via une barre de recherche textuelle sur le nom du client.

## Détails fonctionnels

- Barre de recherche avec placeholder "Rechercher un client..."
- Filtrage en temps réel sur le nom du client (insensible à la casse)
- La recherche s'applique uniquement à la vue courante (actifs ou archivés)

## Critères de done

- [ ] Barre de recherche présente en haut du tableau
- [ ] Filtrage en temps réel fonctionnel
- [ ] Résultats cohérents avec la vue active (actifs ou archivés)
