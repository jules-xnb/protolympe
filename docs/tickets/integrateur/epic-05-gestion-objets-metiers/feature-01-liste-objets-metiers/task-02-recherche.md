# [5-1-2] Recherche dans la liste des objets métiers

## Description

Permettre à l'intégrateur de filtrer les objets métiers affichés dans le tableau via une barre de recherche textuelle.

## Détails fonctionnels

- Barre de recherche avec placeholder "Rechercher un objet métier..."
- Filtrage en temps réel sur le nom et la description (insensible à la casse)

## Critères de done

- [ ] Barre de recherche présente en haut du tableau
- [ ] Filtrage en temps réel fonctionnel sur nom et description
- [ ] Résultats cohérents avec la vue active
