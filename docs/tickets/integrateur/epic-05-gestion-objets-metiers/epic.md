# [5-0-0] Intégrateur - Gestion des objets métiers

## Description

Permettre à l'intégrateur de créer, configurer et gérer les objets métiers d'un client. Un objet métier est une structure de données configurable rattachée aux entités organisationnelles (ex : un risque, un contrôle, un plan d'action). L'intégrateur définit la structure (champs système et personnalisés), importe des définitions et des instances, et gère le cycle de vie (archivage, restauration, duplication).
